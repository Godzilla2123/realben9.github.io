On the Subject of Binary Keys

Hey! I thought this was [realben9.github.io], not KTaNE!

  • Along with normal keys, you may have also seen some stranger looking keys.
  • These are binary keys.
  • Instead of 3 bytes in hexadecimal, these keys are 3 bytes in binary.
  • You must convert these into hexadecimal to leave you with a normal key.
  • Binary keys can also be reversed, as shown by their gradient.
  • In this case, you must reverse the binary before converting to hexadecimal.
  • If you havent seen any yet, there are some examples below.

[EXAMPLE]

A Normal Key - 7fb8e3

A Binary Key - 011111111011100011100011

A Reversed Binary Key - 110001110001110111111110

  • All of these keys should give the same result of "7fb8e3".
  • When you believe you have found and converted all of the keys, you must use them in the same way you did before.
